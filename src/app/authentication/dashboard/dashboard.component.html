ขั้นตอนการใช้งาน

  1. จัดการข้อมูลส่วนตัว
    • กรอกแก้ไขข้อมูลส่วนตัว
    • กรอกข้อมูลสถานศึกษาที่สังเกตการสอน
    • สถานศึกษาที่ทำการฝึกสอน
    • ข้อูมูลครูอาจาร์ยที่นักศึกษาทำการติดต่อ
  2. การสังเกตการสอน
    • เพิ่มข้อมูลการสังเกตการสอน (รายวิชาที่สังเกต เรื่องที่สอน ระดับชั้น จำนวน ชม. ฯลฯ)
      • **นักศึกษาจำเป็นจะต้องสังเกตการสอนเป็นเวลาขั้นต่ำ 12 ชม.
    • ให้ครูผู้สอนทำการยืนยันกับระบบในการสังเกตการสอน
  3. การฝึกประสบการณ์วิชาชีพครู
    • เพิ่มไฟล์ตารางสอนในภาคการศึกษานั้น (ในเมนู จัดการข้อมูลส่วนตัว -> เพิ่มข้อมูลตารางสอน)
    • เพิ่มข้อมูลการฝึกสอนโดยระบุ วันที่สอน วิชา เรื่องที่สอน ฯลฯ
    • แนบลิงก์ การสอน live สดจาก youtube
  4. การทำวิจัยในชั้นเรียน
    • เพิ่มไฟล์ วิจัยในชั้นเรียน และหัวข้องานวิจัยในแต่ละบท
      • **ในระหว่างฝึกประสบการณ์วิชาชีพครูนักศึกษาจะต้องจัดทำวิจัยในชั้นเรียนจำนวน 5 บท
    • (โดย วิจัยในชั้นเรียน จะผ่านการตรวจโดย อาจารย์ หรือผู้ดูแลระบบ เท่านั้น)
  5. การทำงานของระบบส่วนอื่น ๆ
    • ส่วนจัดการสมาชิก
    • ส่วนไฟล์ทั้งหมดในระบบ
    • ส่วนประเมินผลการสอน
    • ส่วน Comment

Features

เว็บไซต์นี้พัฒนาขึ้นด้วย Angular Framework มีลักษณะแบ่งส่วนหน้าเว็บต่าง ๆ เป็นโมดูลจึงง่ายต่อการพัฒนาร่วมกัน ระหว่างนักพัฒนา Front-end และ Back-end

อีกทั้งเว็บไซต์นี้ยังใช้รูปแบบสถาปัตยกรรม Respresentational state transfer Application Program Interface หรือ REST API ที่ใช้ประโยชน์จากเทคโนโลยี Web protocol เพื่อใช้งานในการสร้าง Web Service ซึ่งทำให้ระบบประกอบไปด้วย
